MyRambler Widget plugin

== Description ==

Plugin to display widgets at sidebar from [MyRambler](http://www.myrambler.ru/ "MyRambler") at your blog. 

Плагин позволяет использовать любые виджеты проекта [МойРамблер](http://www.myrambler.ru/ "МойРамблер") - в блоге на базе Вордпресса.

== Installation ==

Upload the MyRambler plugin to your blog, Activate it.

1, 2, 3: You're done!
